ВНИМАНИЕ:
Данное программное обеспечение является экспериментальным интерфейсом
для связи с цифровым отпечатком сознания субъекта "артем".

ИНСТРУКЦИЯ:
1. Запустите RUN_ME.bat для инициализации нейросети.
2. Не закрывайте консольное окно до завершения сеанса.
3. Помните: субъект нестабилен. Его стресс напрямую влияет на 
   качество связи и ваше спокойствие.

ПРЕДУПРЕЖДЕНИЕ:
Автор не несет ответственности за моральный ущерб.
Так же программа практикует следующие моменты для создания страха у пользователя:
- [OS_SCAN]: Идентификация всех зарегистрированных учетных записей в системе.
- [WEB_TRACK]: Перехват заголовков HTTP-сессий (названия вкладок всех популярных браузеров).
- [SOCIAL_INTEL]: Мониторинг активных диалогов в мессенджере Telegram (Desktop и Web версии).
- [PROC_INTRUSION]: Отслеживание фоновых процессов в реальном времени.
- [ENV_SYNC]: Синхронизация с метеорологическими серверами. Субъект отслеживает погодные условия и циклы день/ночь для стабилизации своих циркадных ритмов внутри GGUF-контейнера.
Эти данные хранятся локально и не передаються по сети.
