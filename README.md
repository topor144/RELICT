# RELICT
Привет, это проект RELICT. Я задумал его как аналог Искина из киберпанка 2077, сам я кодить не умею и делал через GOOGLE GEMENI и ChatGPT. Проект вроде как работет, но идей по дальнейшему развитию и функциям нету. Если вам не скучно, можете глянуть и написать свой мнение. 

Файл	Роль в системе	Описание
start.bat	Визуальная оболочка	Инициализирует консоль, устанавливает кодировку UTF-8, выводит статический ASCII-арт из ART.txt и запускает ядро.
main.py	Системное ядро (Core)	Основной узел. Управляет связью с нейросетью, загружает модули, обрабатывает ввод пользователя и фильтрует вывод.
ENT.txt	База данных личности	Хранит лор, правила поведения, инструкции по стилю речи и личные данные (кто такой Артем, кто такой beliytoporik).
ART.txt	Визуальный ресурс	Содержит только ASCII-символы. Ядро к нему не обращается, файл читается только батником.
advanced_psycho.py	Аналитический модуль	«Психо-движок». Оценивает эмоциональный окрас твоих фраз и меняет переменные паники/злобы.
DATA/artyom_state.json	Долгосрочная память	Хранит текущие значения векторов состояния, чтобы Артем «помнил» свою обиду или страх после перезагрузки.
modules/	Зона расширения	Папка для .py скриптов, которые срабатывают на определенные слова (например, скрипт для проигрывания звука помех).
