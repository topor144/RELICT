# RELICT
Привет, это проект RELICT. Я задумал его как аналог Искина из киберпанка 2077, сам я кодить не умею и делал через GOOGLE GEMENI и ChatGPT. Проект вроде как работет, но идей по дальнейшему развитию и функциям нету. Если вам не скучно, можете глянуть и написать свой мнение. 

Файл	Роль в системе	Описание:

start.bat	Визуальная оболочка	Инициализирует консоль, устанавливает кодировку UTF-8, выводит статический ASCII-арт из ART.txt и запускает ядро.
main.py	Системное ядро (Core)	Основной узел. Управляет связью с нейросетью, загружает модули, обрабатывает ввод пользователя и фильтрует вывод.
ENT.txt	База данных личности	Хранит лор, правила поведения, инструкции по стилю речи и личные данные (кто такой Артем, кто такой beliytoporik).
ART.txt	Визуальный ресурс	Содержит только ASCII-символы. Ядро к нему не обращается, файл читается только батником.
advanced_psycho.py	Аналитический модуль	«Психо-движок». Оценивает эмоциональный окрас твоих фраз и меняет переменные паники/злобы.
DATA/artyom_state.json	Долгосрочная память	Хранит текущие значения векторов состояния, чтобы Артем «помнил» свою обиду или страх после перезагрузки.
modules/	Зона расширения	Папка для .py скриптов, которые срабатывают на определенные слова (например, скрипт для проигрывания звука помех).

Свод функций ядра (main.py):

Внутри ArtyomCoreV6 реализованы следующие механизмы:

_load_modules(): Сканирует папку modules/. Если находит там скрипты с функцией execute, подготавливает их к запуску. Это позволяет добавлять новые функции без правки ядра.

get_lore(): Динамическое чтение ENT.txt. Позволяет обновлять лор персонажа «на лету» без перезагрузки программы.

clean_text(text):

Защита имени: Принудительно заменяет любые вариации имени на beliytoporik.

Языковой барьер: Блокирует латиницу. Если нейросеть пытается выдать техническую ошибку на английском, функция заменяет её на лорный текст о «рассыпающихся словах».

generate(u_input, decision):

Формирует финальный промпт для LLM.

Склеивает данные из ENT.txt, текущие векторы состояния (panic, malice) и название активного окна в твоей Windows.

Управляет «температурой» генерации (чем выше паника, тем более хаотичным становится текст).

print_artyom(response): Эмулирует печать человека (или старого терминала), выводя текст посимвольно с задержкой.

3. Функции Психо-движка (advanced_psycho.py)
perceive(text, context): Анализирует твой ввод на ключевые слова.

update_vectors(): Математически изменяет значения в диапазоне от 0.0 до 1.0. Например, если ты часто упоминаешь «октябрь», вектор trauma будет расти.

save_state() / load_state(): Записывает текущие цифры в JSON, чтобы «личность» не обнулялась.
